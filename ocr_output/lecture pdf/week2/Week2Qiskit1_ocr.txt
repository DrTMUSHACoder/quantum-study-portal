# Qiskit

- Open-source Quantum Software Development Kit
- High Performance SDK supporting static, dynamic and scheduled quantum circuits, operators and primitives.
- Leaner - removing meta package architecture
- Qiskit C Api - essential for realizing quantum-centric supercomputing
- Refactoring the core of Qiskit to Rust led to average 16x speed increase
- The shift to Rust not only brings performance gains—It enables a C API to support hybrid compute architectures.
- https://quantum.cloud.ibm.co
